## New

- **Full visual redesign** — Workspaces, boards, and task cards now use a consistent glass-card design language (gradient hairline borders, hover states, priority accent colors) across the whole app, matching the marketing site's look.
- **Workspace Archive & Trash** — Archive a workspace to hide it without losing anything, or move it to Trash (recoverable) instead of an instant, irreversible delete. Both get their own page to browse and restore from.
- **Editable task assignees** — Assign one or several teammates to a task directly from the task detail panel, picked from real workspace members.
- **Team management page** — Invite teammates through an in-place modal, see real-time online/offline presence, and manage pending invitations without leaving the page.
- **Redesigned task creation** — A single "＋" action opens a full task form (title, description, status, priority, due date, labels) instead of a bare inline input.
- **Notification Center filters** — Filter your notification feed by type, with real per-type icons and colors.
- **Dashboard "Recent Activity"** — A live feed of your own actions across every workspace (board/task changes, workspace events), not just things that happened to you.
- **Toast notifications** — Every action now gives instant feedback in the top-right corner.
- **Landing page recognizes you** — Signed-in visitors see a "Dashboard" button and their account menu instead of Sign in/Get started, and the Axiom logo always takes you back to the public site.

## Improved

- Board columns are always the same height and evenly sized, with a visible drop zone even when empty.
- Every board template (Kanban, Bug Tracking, Scrum) now starts with a Backlog column.
- Task activity history shows real teammate and column names instead of raw IDs.
- Full French/English translation coverage across every screen added in this release.
- Faster page-to-page navigation: the app shell's data queries now run in parallel instead of one after another, and every route shows an instant loading state instead of a frozen screen.
- Progressive Web App support (installable, offline fallback page) is back on after being temporarily disabled.

## Fixed

- Assigning a teammate to a task no longer fails silently — a real validation bug rejected every assignment.
- Task cards and the task detail modal now update instantly for everyone when a task is moved, reassigned, or edited, instead of requiring a page reload.
- The Turbopack build root is now pinned, fixing a class of build errors caused by stray lockfiles outside the project.
